# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/featurecreep-cron/piqued/security/advisories/new).

Do not open a public issue for security vulnerabilities.

## Response

You can expect an initial response within 72 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy covers the piqued application code. Issues in upstream dependencies should be reported to their respective maintainers.
